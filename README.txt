To build the task you must copy the Expression Encoder 4 and the IIS Transform Manager assemblies into the libs folder.


Copy the following IIS Transform Manager assemblies into the folder RCE.TransformManager\libs\TransformManager

- Microsoft.Web.Media.TransformManager.Common.dll
- Microsoft.Web.Media.TransformManager.SDK.dll

Copy the following Expression Encoder 4 assemblies into the folder RCE.TransformManager\libs\ExpressionEncoder

- Microsoft.Expression.Encoder.Api2.dll
- Microsoft.Expression.Encoder.dll
- Microsoft.Expression.Encoder.Types.dll
- Microsoft.Expression.Encoder.Utilities.dll
